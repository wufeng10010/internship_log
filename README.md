## 事务
### 概念
是一组操作的集合，它是一个不可分割的工作单位，这些操作要么同时成功，要么同时失败
### 操作
* 开启事务（一组操作开始前，开启事务）
* 提交事务（这些操作全部成功后，提交事务）
* 回滚事务（中间一个操作出现异常，回滚事务） <font color=red size=72>这是事务处理的核心部分！！</font>
